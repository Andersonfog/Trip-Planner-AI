# Trip Planner

An AI-powered travel companion that tells you exactly what to pack for any destination and time of year.

**Live demo:** https://e42f8fc1-157b-4c9b-82d6-040987c19c11-00-1cijyipbbj3ho.riker.replit.dev

## What it does

Trip Planner generates a smart, personalized packing list tailored to your destination and travel dates. Tell it where and when you're going, and the app factors in the local climate and seasonal conditions — typical temperatures, rainfall, and what the weather is usually like that time of year — to recommend exactly what to bring, from rain gear to layers, so you're never caught unprepared.

## Stack

- **Frontend:** React
- **Backend:** Node.js / Express
- **AI:** Claude API (Anthropic)
- **Data:** JSON climate & packing database (27+ destinations, 15 trip types, 92 reference entries)
- **Hosting:** Replit
